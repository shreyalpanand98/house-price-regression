# House Price Prediction using Linear Regression

## Project Overview
This project demonstrates a beginner-friendly implementation of linear regression
to predict house sale prices using the Kaggle House Prices dataset. The focus is on
understanding regression fundamentals, feature selection, and model evaluation.

## Objective
Build and evaluate a linear regression model to predict house prices and analyze
how adding numeric features improves model performance.

- RMSE provides an intuitive measure of prediction error
- RÂ² explains how much variance the model captures
